# ⚔️ Codeforces Problem Solutions

Welcome to my Codeforces solutions repository! This repository is a collection of problems I've solved on **Codeforces** while practicing competitive programming and participating in contests.

## 📖 About

The goal of this repository is to document my learning journey, improve algorithmic thinking, and build consistency in solving programming challenges.

Every solution is written with correctness, readability, and efficiency in mind.

## 🛠️ Language Used

* C++17

## 📁 Organization

The repository contains accepted solutions for various Codeforces problems. Files are named using the problem number, contest ID, or problem title whenever applicable, making them easy to locate.

## 🚀 Running the Solutions

Compile the source file:

```bash
g++ -std=c++17 solution.cpp -o solution
```

Run the program:

```bash
./solution
```

On Windows:

```bash
solution.exe
```
